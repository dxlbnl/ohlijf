# AI Agent Instructions

> [!IMPORTANT]
> If you are an AI assistant or agent working on this repository, you **MUST** follow these rules.

## The LLM Wiki
This project maintains an active "LLM Wiki" located in the `wiki/` directory. It contains all relevant technical decisions, architecture, and design details.

1. **Query the Wiki First**: Before proposing solutions, making architecture choices, or working on complex details, you MUST query the wiki. Read `wiki/index.md` to orient yourself, and view any relevant pages to gather context.
2. **Maintain the Wiki**: Do not treat the wiki as read-only. If you add a new package integration, change the database schema, or update the tech stack, log the event in `wiki/log.md` and update/create the corresponding pages. Refer to `wiki/schema.md` for specific instructions.
